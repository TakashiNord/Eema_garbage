Данным обновлением выполняется:



ПРИМЕЧАНИЕ:
       ДЛЯ БДТИ С РЕПЛИКАЦИЕЙ выполнить обновление на обоих основных серверах.
       ДЛЯ БДТИ С ВЫДЕЛЕННЫМ СЕРВЕРОМ АРХИВОВ выполнить обновление только на основном(ых) сервере (ах).

-------------------------------------------------------------------------------
ПЕРЕД ВЫПОЛНЕНИЕМ обновлений внести изменения в секцию установок файла main.bat
а именно: 
  строку соединения к БД (алиас из файла tnanames.ora, если не RSDU) ,
  наименование схемы, в котор. вносятся изменения  (RSDUADMIN) 
  и пароль владельца схемы
   
ВЫПОЛНЕНИЕ обновление  - запуском командного файла main.bat 
непосредcтвенно из папки, в котор. развернуто обновление.

ПОСЛЕ ВЫПОЛНЕНИЯ лог-файл работы скрипта обновления появится в этой же папке. 
(Предполагается, что обновление производится на машине-клиенте Oracle под ОС Windows,
sqlplus присутствует всегда, если установлен полный клиент). 

Для справки - последовательность выполнения всех прилагаемых файлов скриптов - см. файл main.sql

-------------------------------------------------------------------------------
Для отображения в командном окне русских букв необходимо 
установить для окны шрифты True Type (обычно Consolas или Lucida Console)
-------------------------------------------------------------------------------

Возможные коды ошибок в логе:

1.
Если обновление уже было выполнено и присутствует запись о нем в таблице регистрации обновлений, 
причем со статусом <не ошибка>, выполнения сценария завершается со следующим сообщением:
 
ORA-20002: Обновление уже выполнено. Выполнение сценария остановлено 
ORA-06512: на  line ... 
это не ошибка, это значит, все уже сделано.
